 Ronda Analytics
 
O Ronda Analytics é um sistema web de gerenciamento de dados que conta com as informações que já são recolhidas pela Ronda Maria da Penha. Através desses dados é possível a geração de relatórios que cruzam diferentes filtros de dados selecionados pelo usuário no portal. Através desse programa é possível que cada participante do Comitê de Governança tenha seu login de acesso exclusivo, que determinará o acesso aos dados permitidos para sua esfera de poder. Cada um com uma permissão de acesso pré definida, pode então partir para a geração de relatórios baixáveis e gráficos, que poderão embasar a tomada de decisão e a criação de políticas públicas direcionadas ao combate à violência contra mulher, fortalecendo assim a rede que atende a participante da Ronda Maria da Penha e facilitando o trabalho dos policiais da Ronda.
 
O Ronda Analytics é um sistema online mas será implementado usando um banco de dados gerado através da automatização do Formulário de Acolhimento, aplicado pelos policiais da Ronda Maria da Penha. Este formulário é hoje atualizado manualmente, mas projetos já estão sendo desenvolvidos com o intuito de automatizar essa aplicação.
 
No portal, após o login cada usuário terá as informações que a sua permissão autoriza. É aberta a tela de início com estatísticas da Ronda no mês corrente. Na barra de menus é possível fazer a seleção de dados desejados para a geração de relatório e gráfico, esses dados são apresentados em forma de gráfico e tabela, fornecendo os dados brutos e tratados em gráficos.
 
A maior motivação de realizar essa aplicação é o fortalecimento do acesso à informação pelos participantes do Comitê de Governança para que políticas públicas possam ser criadas embasadas em dados reais do dia-a-dia da Ronda Maria da Penha. Hoje os dados da Ronda são compilados manualmente e filtrados através de uma planilha em Microsoft Excel. O Ronda Analytics vem para automatizar o trabalho de tratamento dos dados, divulgação dos dados entre os participantes do Comitê e a potabilidade desses dados por meio de downloads, que permitem que sejam transportados pelos usuários autorizados para reuniões, palestras, entrevistas, entre outros.
